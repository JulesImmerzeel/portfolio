Smaddle

Smaddle.nl was een groepsproject for de minor Smart Things die ik in het vierde jaar van mijn opleiding heb gevolgt. Het idee was om in tien weken tijd een startup op te zetten met een werkend prototype voor een product. Een smaddle is een smart fietszadel, en een website waarop de data van dit fietszadel bezichtigt kon worden. Kort samengevat was het een gps-tracking systeem tegen diefstal, maar met een paar extra snufjes zoals knipperlichten. Op de website kon door middel van websockets realtime de exacte lokatie van de fiets getraceerd worden.

Ik heb mij tijdens dit project bezig gehouden met de frontend. Hiervoor heb ik voor het eerste kennis gemaakt met Vue.
Ik vond dit een onwijs leuk en leerzaam project. Mocht ik in de toekomst weer een spa bouwen, dan denk ik dat ik weer voor Vue in combinatie met Vuex zou kiezen.

  • .Net
  • Vue
  • Website

Starling Structures

Starling Structures is een bureau gespecialiseerd in stedebouw en landschap gevestigd in Den Haag. Starling Structures wilde een website waarop potentiële klanten kunnen zien wie zij zijn en wat ze doen. Ik heb de website in Django gemaakt. Django heeft een handige buildin admin functionaliteit waardoor de klant zelf zijn projecten kan managen. Ik heb van dit project veel geleerd over SEO en databases.

World whide board

World whide board was een website waarop iedereen iets kon tekenen. De site update realtime automatisch voor iedereen die online is als er iets op getekend wordt.
Het leek mij een leuk project om meer te leren over webRTC en websockets. De website is momenteel niet online, maar de code kunt u op github zien.

Proefpersonen informatie

Proefpersoneninformatie.nl was mijn stage opdracht. Kort samengevat is het website waar potentiële deelnemers van een klinische studies meer informatie kunnen vinden over een onderzoek. De site is gemaakt met als uitgangspunt dat informatie over klinische onderzoeken voor iedereen begrijpbaar moet kunnen zijn, dus ook voor laaggeletterde en kinderen.

De frontend gebruikt Sass en React, de backend met Django. Tijdens mijn stage heb ik grote stappen gemaakt in mijn programmeer vaardigheden.
Daarnaast heb ik veel geleerd over samenwerken met andere stackeholders, werken met een grote codebase onbekende code base en hoe het is om full time werken :)

  • Django
  • Sass
  • React
  • Website
